# Audition Mate

Your digital reader for self-tapes and rehearsals.

## Features
- Record scene lines with automatic transcription.
- Rehearse with a digital reader that waits for your cue.
- Self-tape mode with a built-in teleprompter.

## Speech Recognition
This app uses the browser's **Web Speech API** for transcription and cue word detection.
- **Internet Connection**: Most browsers require an active internet connection for speech recognition.
- **Supported Browsers**: Chrome, Safari, and Edge are recommended.
- **Fallback**: If speech recognition is not supported in your browser, you can still record audio and manually enter line text.

## Setup
1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Open the app in a supported browser.
